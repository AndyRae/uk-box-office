UK Box Office Data

A live dashboard for UK Box Office data with an extensive archive - as far as I could find, this hasn't been done in the public space at all.

Through using the BFI weekend box office reports, it contains box office gross back to 2007, with new data every week on a Wednesday.

It is not an exhaustive data set - the source only includes the weekly box office for the top 15 films, British films, and new releases. But this still covers 95%+ of the market. There are also inevitably errors in the data, as there are over 40,000 records.
